Parametric Rosettes


This sketch can be used to generate curves created by adding together various circular motions. Each joint of the drawing machine represents a circular motion with a frequency (how fast it spins), amplitude (length of the joint), and phase (starting angle).

Basic Usage: Use the "Preset Pattern" dropdown in the top left to pick one of the preset patterns I made, listed by symmetry type. Spacebar toggles the drawing machine arm.

Advanced Usage: Use the "Custom Pattern" input box to type in frequencies, amplitudes and phases. Then press Enter. Note: phases are listed as multiples of pi
